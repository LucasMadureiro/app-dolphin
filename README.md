# App Dolphin ⚡

O **App Dolphin** é um simulador de recarga inteligente para veículos elétricos, desenvolvido especificamente para motoristas da linha BYD (Dolphin Mini, GS e Plus). O objetivo principal é fornecer estimativas precisas de custo e tempo de carregamento, ajudando na tomada de decisão financeira e logística durante o turno.
link: https://lucasmadureiro.github.io/app-dolphin/

## 🚀 Funcionalidades

- **Simulação de Recarga em Tempo Real**: Ajuste a bateria atual e a meta de carga para ver instantaneamente o kWh necessário.
- **Cálculo de Eficiência Energética**: O sistema aplica automaticamente perdas de calor baseadas na potência do carregador selecionado (120kW, 60kW ou 7kW).
- **Gestão de Tarifas**: Configuração personalizada de valor por kWh e taxa de ativação do eletroposto.
- **Assistente de Voucher**: Alerta inteligente que indica se compensa usar um código promocional (ex: R$ 40,00) ou pagar no cartão.
- **Interface Multi-Tema**: Suporte a temas **Dark**, **Light** e **Neon**, salvos localmente para preferência do usuário.
- **PWA (Progressive Web App)**: Instalável no celular e funciona 100% offline via Service Workers.

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando tecnologias web modernas:

- **HTML5**: Estrutura semântica do aplicativo.
- **CSS3**: Layout responsivo utilizando **CSS Grid** e **Flexbox**, além de variáveis (Custom Properties) para o sistema de temas.
- **JavaScript (ES6+)**: Lógica de cálculo, manipulação do DOM e integração com a API de armazenamento local.
- **PWA (Service Workers & Manifest)**: Para garantir a instalação na tela inicial e resiliência offline.
